About this project

This repository is an experimental workspace for UX research and interface design work supporting Texas A&M University Libraries. It serves as a controlled environment for testing ideas, validating design system component usage, and building shareable prototypes before they reach production.

All work here uses the Aggie UX (AUX) design system — the university-wide component library maintained by the Division of Marketing & Communications — to ensure that any concepts developed here are compatible with TAMU's design standards and can be handed off cleanly.

Project goals

Prototype freely

Build and iterate quickly without the constraints of a production codebase.

Validate AUX usage

Document correct component usage patterns and surface edge cases in the design system.

Explore content UX

Test how different content types — collections, research guides, exhibits — work within AUX layouts.

Shareable artifacts

Produce demo pages and documentation that can be shared with stakeholders and collaborators.
